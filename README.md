# CSharp-Console-Project-Exception-Handling

A simple C# console application developed using .NET 10.0.  
This project demonstrates exception handling and safe user input validation using `TryParse()` and `try-catch` in a calculator program.

---

## 📌 Project Overview

The application takes two numeric inputs from the user and performs basic arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division

The project demonstrates two different approaches for handling invalid input and runtime errors:

- Using `double.TryParse()`
- Using `try-catch` exception handling
